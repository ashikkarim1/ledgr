## SUPPORTING COPY GUIDELINES

When refreshing page copy, emphasize these themes:

1. **"Groundbreaking"** — Use 2-3x per page
   - "Groundbreaking AI that understands UAE regulations"
   - "Groundbreaking cloud architecture built specifically for compliance"

2. **"Eliminate gaps"** — Frame as solving specific problems
   - "Eliminate spreadsheet errors"
   - "Eliminate FTA penalties"
   - "Eliminate compliance burden"

3. **"Intelligent automation"** — Not just "automation"
   - "Intelligent AI" (implies learning, not just scripting)
   - "Smart categorization" (not "automatic" which feels passive)

4. **"Audit-ready"** — Reframe compliance as continuous preparation
   - "Audit-ready every transaction"
   - "Compliance built in, not bolted on"

5. **"One platform"** — Emphasize completeness
   - "FTA + VAT + Payroll in one platform"
   - "One dashboard, complete financial picture"
