Installation

Copy the three core files to your project assets directory:

  • assets/chat-widget.js - Main chat component (45KB minified)
  • assets/escalation-modal.js - Escalation flow (12KB minified)
  • assets/chat-styles.css - Complete styling (28KB minified)

Total bundle size: ~85KB (uncompressed) → ~18KB (gzipped)

Quick Setup

1. Include in your HTML:

<!-- Styles --> <link rel="stylesheet" href="assets/chat-styles.css"> <!-- Mount point --> <div id="chat-app"></div> <!-- Scripts --> <script src="assets/chat-widget.js"></script> <script src="assets/escalation-modal.js"></script>

2. Initialize in your JavaScript:

const chatWidget = new ChatWidget({ apiBaseUrl: '/api/v1/chat', whatsappNumber: '+971501234567', enableDarkMode: true, position: 'bottom-right' }); chatWidget.mount('#chat-app'); const escalationModal = new EscalationModal({ whatsappNumber: '+971501234567', ticketApiUrl: '/api/v1/escalate' }); escalationModal.mount('#chat-app');

Configuration Options

Option Type Default Description
apiBaseUrl string /api/chat AI service endpoint
whatsappNumber string +971501234567 Escalation WhatsApp number
enableDarkMode boolean true Enable dark mode toggle
maxHistoryItems number 50 Max saved conversations
position string bottom-right bottom-right or bottom-left
primaryColor string #FF5C00 Brand primary color

API Integration

Chat Endpoint: POST /api/v1/chat

{ "message": "How do I set up VAT?", "sessionId": "session-1234567890", "context": [], "streaming": true }

Response: Server-sent events stream with message content

Escalation Endpoint: POST /api/v1/escalate

{ "summary": "Need help with invoice discounting", "contactTime": "asap", "priority": "standard", "contactMethod": "whatsapp", "sessionId": "session-1234567890", "context": [...] }

Response:

{ "success": true, "ticketId": "TKT-123456", "estimatedWaitTime": "15 minutes", "agentName": "Ahmed" }
